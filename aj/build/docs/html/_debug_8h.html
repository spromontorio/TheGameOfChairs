Typedef Documentation

typedef void(* QCC_DbgMsgCallback)(DbgMsgType type, const char *module, const char *msg, void *context)

Prototype for the debug message callback.

This callback allows application code to recieve debug message rather than the debug messages going to STDERR or a file.

Parameters
typeType of debug message. One of the DbgMsgType enumerations.
moduleA C string identifying the module that the debug message is from.
msgA C string containing the debug message.
contextAn opaque pointer to context data provided by the application when registering the callback.
