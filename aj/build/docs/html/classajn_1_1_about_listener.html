handler for the org.alljoyn.About.Anounce sessionless signal

The objectDescriptionArg contains an array with a signature of a(oas) this is an array object paths with a list of interfaces found at those paths

The aboutDataArg contains a dictionary with with AboutData fields that have been announced.

These fields are

  • AppId
  • DefaultLanguage
  • DeviceName
  • DeviceId
  • AppName
  • Manufacturer
  • ModelNumber

The DeviceName is optional an may not be included in the aboutDataArg. DeviceName is required for System Apps.

DeviceName, AppName, Manufacturer are localizable values. The localization for these values in the aboutDataArg will always be for the language specified in the DefaultLanguage field.

Parameters
[in]busNamewell know name of the remote BusAttachment
