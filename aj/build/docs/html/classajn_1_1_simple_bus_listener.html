Detailed Description

Helper class that provides a blocking API that allows application threads to wait for bus events.

QStatus ajn::SimpleBusListener::WaitForEvent ( BusEvent &  busEvent,
uint32_t  timeout = FOREVER 
)

Wait for a bus event.

Parameters
busEventReturns the busEvent type and related information.
timeoutA timeout in milliseconds to wait for the busEvent, 0 means don't wait just check for an busEvent and return, FOREVER (-1) means wait forever.
Returns
  • ER_OK if an even was received.
  • ER_TIMEOUT if the wait timed out.
  • ER_ALERTED_THREAD if the wait unblocked due to a signal
