Detailed Description

template<typename T>
class qcc::ManagedObj< T >

ManagedObj manages heap allocation and reference counting for a template parameter type T.

ManagedObj<T> allocates T and sets its reference count to 1 when it is created. Each time the managed object is passed by value or otherwise copied (which is an inexpensive operation), the underlying heap allocated T's reference count is incremented. Each time a ManagedObj instance is destructed, the underlying T reference count is decremented. When the reference count reaches zero, T itself is deallocated using delete.

template<typename T>
qcc::ManagedObj< T >::ManagedObj ( const ManagedObj< T > &  other,
bool  isDeep 
)
inline

Create a copy of managed object T.

If isDeep is true Create a deep (clone) copy of a managed object. A ManagedObject created using this constructor copies the underlying T object and wraps it in a new ManagedObject with 1 reference.

if isDeep is false Do not make a deep copy of the managed object instead make a new reference to the existing object and increment the reference counter by +1.
