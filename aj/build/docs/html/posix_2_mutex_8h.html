Define a class that abstracts Linux mutex's.

Macro Definition Documentation

#define MUTEX_CONTEXT   __FILE__, __LINE__

a macro that can be passed into the Mutex::Lock/Unlock member functions to help when debugging Mutex related issues.

When running in debug mode, this will cause the code to log the name of the file and the line number of that file each time a Mutex lock is obtained and released. Logging must be turned on to see this information.
