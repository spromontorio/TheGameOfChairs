Detailed Description

Abstract base class implemented by AllJoyn users and called by AllJoyn to inform users of bus related events.

TransportMask  transport,

Called by the bus when an external bus is discovered that is advertising a well-known name that this attachment has registered interest in via a DBus call to org.alljoyn.Bus.FindAdvertisedName.

Parameters
